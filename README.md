## Or to start a project:
* **Clone repository** 
  * $ git clone **git@github.com:radvam/songbird.git**
* **Go to project folder in git-bash** 
* **Go to branch gongbird** 
  * $ git checkout gongbird  
* **Install Node modules** 
  * $ npm i 
* **Start project**
  * $ npm start
